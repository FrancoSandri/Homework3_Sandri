## Comandos para compilar y ejecutar el código en C++

1. Cambiar el directorio:
   ```sh
   cd ej1
   ```

2. Compilar el código:
   ```sh
   make
   ```
3. Ejecutar el programa:
   ```sh
   ./main.exe
   ```
4. Borrar los archivos creados al compilar:
    ```sh
   make clean
   ```
